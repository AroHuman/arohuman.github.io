Compatibility

  • Compatible with iOS 11.0 +

Description

  • Revert to the original video quality selector in YouTube app.

Changelog

  • 1.0.1: Fixed crashing in YouTube 16.20.5

  • 1.0.0: Initial release

Source Code

  • #import <Foundation/Foundation.h>

@interface YTVideoQualitySwitchOriginalController : NSObject
- (instancetype)initWithParentResponder:(id)responder;
@end

%hook YTVideoQualitySwitchControllerFactory

- (id)videoQualitySwitchControllerWithParentResponder:(id)responder {
	Class originalClass = %c(YTVideoQualitySwitchOriginalController);
	return originalClass ? [[originalClass alloc] initWithParentResponder:responder] : %orig;
}

%end
