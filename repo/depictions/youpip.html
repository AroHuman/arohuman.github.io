Compatibility

  • Compatible with iOS 11.0 +

Description

  • Enable native PiP in YouTube app.

    If you encounter playback speedup issue for your PiP, please be informed that this is the issue of iOS itself and not YouPiP. If Apple could fix this in the future, YouPiP would try to imitate their fix in an update.

    If you encounter "No stream Tap to Retry" error for your video, you are recommended to upgrade YouTube to the latest version.

    The changelogs are annotated with the versions of YouTube app that YouPiP supports.
